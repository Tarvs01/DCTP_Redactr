Tidbits about the app

  • If the redact symbol is one character, it will replace each letter of the redacted word with that character
  • If the redact symbol is more than one character, it will replace the redacted word with that symbol
  • If you want to redact a word with a prefix punctuation such as sleep in "sleep", put a space between the prefix punctuation and the first letter. That is, "sleep" becomes " sleep"
  • In the output, the redacted words are colored blue to show their location
  • Javascript is pretty fast. So if you want to see a higher time, enter a larger text into main content
